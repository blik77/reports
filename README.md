# reports
Сайт для просмотра данных в различных форматах по продуктам НПЗ, металлам и агрикультурам.

Frontend создан с помощью библиотеки Ext JS (Sencha), JQuery, HightCharts, D3.js, TinyMce, Bootstrap.
<br>
Backend создан на Java, но не приведен из-за NDA.
